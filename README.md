# Simple Python Calculator

A basic calculator program written in Python that can perform addition, subtraction, multiplication, and division.  

## Features

- Addition (`+`), Subtraction (`-`), Multiplication (`*`), Division (`/`)
- Input validation for numbers
- Prevents division by zero
- User can perform multiple calculations in a loop
- Option to quit anytime by entering `Q`

## How to Use

1. Download the `calculator.py` file.
2. Open a terminal and navigate to the project directory.
3. Run the calculator using Python:

```bash
python calculator.py
